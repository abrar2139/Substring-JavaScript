The substring() method in JavaScript retrieves a portion of a string based on specified start and end indices, returning a new string. It requires two parameters: the starting index (inclusive) and the ending index (exclusive). Providing only one parameter treats it as the start index, extracting characters from there to the string's end. Negative indices are interpreted as counting from the string's end. Unlike substr(), substring() doesn't allow negative start positions. This method does not alter the original string. It's commonly used for extracting substrings for various string manipulation tasks.
